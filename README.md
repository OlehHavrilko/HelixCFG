# HelixCFG

## Overview
HelixCFG is a comprehensive configuration and management system that combines 3D visualization, domain-driven architecture, and data management capabilities. The project is designed to provide a flexible, scalable platform for configuration management with an intuitive user interface.

## Key Features
- **Domain-Driven Architecture**: Core business logic separated into model, engine, and service layers
- **3D Visualization**: Powered by Three.js for interactive 3D rendering
- **Modern UI**: Clean and responsive user interface
- **Data Management**: Comprehensive data layer with catalog, pricing, and preset management
- **Extensible Design**: Modular architecture for easy feature addition
